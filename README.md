# ascii-comment 

Convert comment, selection or whole document to ASCII characters.

## Features

For example:

```
// moje úžasná funkce
function myAwesomeFunction () {}
```

Will be converted into:

```
// moje uzasna funkce
function myAwesomeFunction () {}
```
